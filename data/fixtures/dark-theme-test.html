Dark Theme Test Page

Section One

This page uses a native dark color scheme. The background is #0d1117 and text is #e6edf3.

It does not rely on prefers-color-scheme media queries — the dark palette is applied unconditionally.

Section Two

Useful for testing how Surfingkeys UI elements (hints, popup, keystroke bar) look on a dark page.

Check that the link color and hint labels are legible against the dark background.

Section Three

Additional content to make the page scrollable for scroll-related command tests.
