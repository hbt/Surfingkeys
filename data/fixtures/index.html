This is the fixtures server for Surfingkeys CDP tests.

Server: tests/fixtures-server.js
Port: 9873

Available Fixtures

The following test pages are available:

  • hackernews.html - Hacker News clone for link hints testing
  • index.html - This page

Usage

Start the fixtures server:

node tests/fixtures-server.js

Then access test pages at:

http://127.0.0.1:9873/
http://127.0.0.1:9873/hackernews.html

For Developers

The server automatically:

  • Serves files from data/fixtures/
  • Handles root path (/) by serving index.html
  • Sets proper Content-Type headers
  • Enables CORS for all origins
  • Validates that all served files stay within the fixtures directory
